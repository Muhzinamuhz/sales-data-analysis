# 📊 Sales Data Analysis Project

This project focuses on analyzing a retail dataset using Microsoft Excel.

## 🔹 Tools Used
- MS Excel  
- Pivot Tables  
- Charts & Dashboard  

## 🔹 Key Tasks Performed
- Cleaned and prepared dataset for analysis  
- Used Pivot Tables to extract insights  
- Built an interactive dashboard using charts  
- Analyzed sales trends and performance  

## 🔹 Key Insights
- Identified top-performing products and regions  
- Observed sales trends across different categories  
- Supported data-driven decision-making
